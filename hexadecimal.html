Combinando: 1111111111111111111111111₂

Octal:

Método 1: Convertir primero a binario, luego agrupar de 3 en 3 bits desde la derecha:

001 111 111 111 111 111 111 111 111 → 1 7 7 7 7 7 7 7 7 = 17777777₈

Decimal:

  • F x 16⁶ = 15 x 16777216 = 251658240
  • F x 16⁵ = 15 x 1048576 = 15728640
  • F x 16⁴ = 15 x 65536 = 983040

  • F x 16¹ = 15 x 16 = 240
  • F x 16⁰ = 15 x 1 = 15

Suma total = 251658240 + 15728640 + 983040 + 61440 + 3840 + 240 + 15 = 268435455

2. Hexadecimal: A90456₁₆

  • A → 1010
  • 9 → 1001
  • 0 → 0000
  • 4 → 0100
  • 5 → 0101
  • 6 → 0110

Combinando: 101010010000010001010110₂

  • 9 x 16⁴ = 9 x 65536 = 589824

Suma total = 10485760 + 589824 + 0 + 1024 + 80 + 6 = 11076694

  • 4 x 16⁰ = 4 x 1 = 4

Suma total = 4096 + 0 + 32 + 4 = 4132

Binario:

  • 4 → 0100
  • 9 → 1001
  • 8 → 1000
  • 0 → 0000

Eliminando los ceros no significativos: 100100110000000₂

000 100 100 110 000 000 → 0 4 4 6 0 0 = 44600₈

Eliminando los ceros no significativos: 44600₈

  • 9 x 16² = 9 x 256 = 2304
  • 8 x 16¹ = 8 x 16 = 128
  • 0 x 16⁰ = 0 x 1 = 0

Suma total = 16384 + 2304 + 128 + 0 = 18816
