Conversión de Números Octales

1. Octal: 357₈

Binario:

Convertimos cada dígito a su equivalente binario de 3 bits:

  • 3 → 011
  • 5 → 101
  • 7 → 111

Combinando: 011101111₂

Decimal:

  • 3 x 8² = 3 x 64 = 192
  • 5 x 8¹ = 5 x 8 = 40
  • 7 x 8⁰ = 7 x 1 = 7

Suma total = 192 + 40 + 7 = 239

Hexadecimal:

0000 1110 1111 → E F = EF₁₆
